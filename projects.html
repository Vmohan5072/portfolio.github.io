MythBot: A Discord Bot for League of Legends

Developed in JavaScript with the Discord.js framework, MythBot communicates with the Riot Games API to provide League of Legends account information and management to Discord servers. Features include displaying player ranks, top champion mastery, and creating queues for custom lobbies. Hosted on Railway.

Naive Bayes Spam Filter

A Python-based web application hosted on Heroku that uses the Naive Bayes classifier to filter spam emails. It processes text files and determines the likelihood of them being spam using a pre-trained model from the Enron Spam Dataset.
